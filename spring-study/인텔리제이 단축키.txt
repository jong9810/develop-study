# Intellij 단축키
0) 
F2 : 빨간 줄(에러)이 쳐진 곳으로 이동
컬렉션.iter :  컬렉션 순회하는 for()문 자동 생성

1) ctrl
ctrl + b / ctrl + click : 클래스명 부분에서 사용하면 해당 클래스가 구현된 파일로 이동.
ctrl + e : 최근에 봤던 파일 목록
ctrl + d : 아래줄에 해당 코드를 복사한다.
ctrl + y : 현재 커서가 있는 줄을 삭제한다.
ctrl + o : override methods

2) alt
alt + 1 : project 탭을 열고 닫는다.
alt + 4 : run 탭을 열고 닫는다.
alt + f12 : 터미널 열고 닫기
alt + Enter : import, static import, 오류 해결 방안 제시, implement methods, 
alt + insert : 생성자, getter/setter, 오버라이딩 등을 할 수 있는 메뉴를 열어준다.
alt + j : 같은 단어 찾기


3) shift
shift + f6 : 변수명, 파일명을 일괄적으로 변경할 때 사용한다.
shift + f10 : 이전 실행했던 파일을 재실행해준다.

4) ctrl + shift
ctrl + shift + Enter : 문장 완성(세미콜론, 괄호닫기 등).
ctrl + shift + 방향키 : 코드를 방향키 방향으로 이동시킨다.
ctrl + shift + f : 파일 검색
ctrl + shift + t : 클래스에 있는 메소드들에 대해 테스트 코드의 틀을 만들어준다.

5) ctrl + alt
ctrl + alt + s : setting 창 열기
ctrl + alt + insert : 새 파일을 생성(선택된 디렉토리 안에)
ctrl + alt + n : 변수를 생성하고 그 변수를 리턴하는 경우 한 라인으로 합쳐준다.
ctrl + alt + v : 해당하는 반환값으로 변수를 생성해준다.
ctrl + alt + m : 특정 부분을 드래그 한 후에 사용하면 해당 부분을 메소드로 따로 만들어준다.
ctrl + alt + b : 해당 인터페이스를 구현한 구현체 목록을 보여준다.

6) alt + shift
alt + shift + click : 다중 커서 생성
alt + shift + j : 같은 단어 찾기 취소

7) ctrl + alt + shift
ctrl + alt + shift + t : refactor 메뉴(inline : 변수 선언, 리턴 합치기)
